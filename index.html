تعليم Termux 💻

دليلك السريع لأهم أوامر Termux على الأندرويد

البداية مع Termux

  • شنو Termux؟ تطبيق طرفية لينكس للأندرويد بدون روت
  • أول أمر لازم تكتبه: termux-setup-storage عشان يسمح بالوصول للملفات
  • تحديث الحزم: pkg update && pkg upgrade

انسخ والصق في Termux:

pkg update -y && pkg upgrade -y

أوامر الملفات والمجلدات

الأمر الوظيفة
ls عرض الملفات
ls -a عرض الملفات المخفية
cd اسم_المجلد الدخول لمجلد
cd .. الرجوع خطوة
mkdir اسم إنشاء مجلد جديد
rm اسم_الملف حذف ملف
rm -rf اسم_المجلد حذف مجلد كامل
pwd معرفة مكانك الحالي

تثبيت الحزم والبرامج

pkg - مدير الحزم الأساسي

pkg install python - تثبيت بايثون

pkg install git - تثبيت Git

pkg search اسم - البحث عن حزمة

أشهر الحزم اللي تحتاجها

pkg install python nodejs php git nano vim wget curl

ملاحظة: استخدم -y عشان يوافق تلقائياً مثل: pkg install python -y

أدوات مهمة وسكربتات

  • محاكاة Kali: pkg install root-repo && pkg install metasploit
  • سيرفر محلي: pkg install apache2 ثم apachectl start
  • محرر نصوص: nano file.txt أو vim file.txt
  • تحميل ملفات: wget رابط_الملف
  • استنساخ من GitHub: git clone رابط_المستودع

تثبيت Python وتشغيل سكربت:

pkg install python -y && python script.py
